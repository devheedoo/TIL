# Vue 3

## 1강: Vue 3 버전 설치랑 셋팅

- CLI 설치: `$ npm install -g @vue/cli`
- VS Code Extension: Vetur, HTML CSS Support, Vue 3 Snippets
- 프로젝트 생성: `$ vue create vuedongsan`
- 첫 시작 파일: `src/App.vue`
- 기본 코드 구성
  - `<template>`: HTML
  - `<script>`: JavaScript
  - `<style>`: CSS
- 미리보기: `$ npm run serve`
